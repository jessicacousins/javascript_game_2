# Pizza Menu

## Description

This interactive flip menu displays the different order items for a pizza restaurant.

## Dependencies

- Bootstrap
- Google Fonts: 'Abril Fatface'
- `npm install express mongoose body-parser cors`

## Backend

Run the backend server with:

```
node server.js
```
